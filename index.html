Below are some Physical Text Styles

Bold
b tag is used to bold the text
Italic
i tag is used to italicize the text
Underline
u tag is used to underline the text
Strike Through
strike tag is used to place a cut on the line
Small
small tag is used to reduce the size of a text
Big
Big tag is used to increase the size of a text
Subscript
sub tag will the write the content in the foot of a text; for eg H2O
Superscript
sup tag will the write the content in the power of a text; for eg x2
Teletype
tt tag will the write the content in the typewrite writing
Conclusion of physical tags

Here are some examples of physical text styles in HTML: When you want to emphasize something, you can make it bold. If you're trying to add emphasis with a softer touch, italicizing is a good option. To show important information, you might want to underline certain words. When referencing outdated information, you can use a strike-through to show it's no longer valid. For footnotes or smaller text, use the small tag, while for more prominent text, the big tag can be useful. In chemistry, we write water as H2O, utilizing the subscript tag. In mathematics, you might see a formula like x2 + y2 = z2, where superscript is used. Finally, the teletype style gives text a typewriter-like appearance, which is less common but useful for representing code snippets in legacy systems.
